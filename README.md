# ByeScreenTime (神)
この.mobileconfig（iPhone/iPad/iPodのプロファイル）はスクリーンタイムを無効にします。それには、デバイスを「偽の監視」状態にする必要があります。そのためにはiMazing、Nugget、MisakaX、iSupervisor（脱獄している場合）などのツールが有用です。
## この機能はiOSデバイスの監視なしでは動作しません！(偽の監視でOK)
iOS 12～18でのみテスト済みです！iOS 18以降ではテストしていませんが、ほとんど動作するはずです。


## インストール
1. [このリンク先のファイルを保存](https://raw.githubusercontent.com/singlekeycap/ByeScreenTime/master/ByeScreenTime.mobileconfig)。
2. (ポップアップを許可する。)
3. 設定を開き、`VPNとデバイス管理` → `ダウンロード済みのプロファイル` をクリック。
4. インストールする。
5. スクリーンタイムを無効化
